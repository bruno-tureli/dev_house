Desenvolvi um projeto pessoal onde criei uma API REST utilizando Node.js com Express, implementando operações completas de CRUD para gerenciamento de dados.

Para persistência, utilizei o MongoDB Atlas, o que me permitiu trabalhar com banco de dados em nuvem e entender melhor a integração entre aplicação backend e banco NoSQL. Realizei operações como inserção, consulta, atualização e exclusão de registros diretamente no banco.

Para testes e validação dos endpoints, utilizei o Insomnia, onde consegui simular requisições HTTP (GET, POST, PUT e DELETE) e verificar o comportamento da API em diferentes cenários.
